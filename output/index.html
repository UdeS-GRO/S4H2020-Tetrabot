class main.Controler¶

Class to control parameters of the walk sequence and simulation

Parameters

  • step_index – the index of the current step in the simulation

  • resolution – the resolution of the simulation

  • period – the period of the resolution of the simulation

  • delay – the pause between each walking sequence

class gui.Gui¶

Class for the Graphics user interface, where the simulation is displayed along with visualization options.

Parameters

  • Label – label of the window

  • and window_height (window_width) – Visualization window size

  • Buttons – Multiple buttons that the user can set for different actions (Standing, Run, Stop)

  • Checkbuttons – Multiple checkbuttons that the user can set for different visualizations (3D, Draw_trajectory)
